# Email Auth MVC (PHP & MySQL)

## Description
A secure email-based authentication system built using PHP and MySQL, following MVC architecture.  
Includes: user registration, email verification, OTP login, and home page.

## Requirements
- PHP 8.x
- MySQL / MariaDB
- Composer
